<h1 align="center">
  <p>Jest TODO Reporter</p>
</h1>
Report todo tests in Jest

## Installation

Using yarn:

```bash
$ yarn add -D jest-todo-reporter
```

Using npm:

```bash
$ npm i -D jest-todo-reporter
```

## Usage

Jest CLI:

```bash
jest --reporters jest-todo-reporter
```

Jest Config:

```json
{
  "reporters": ["jest-todo-reporter"]
}
```
